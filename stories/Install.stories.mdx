## 安装

使用 npm 或 yarn 安装

<div>npm install srm-ui --save</div>
<div>yarn add srm-ui</div>

## 浏览器引入

在浏览器中使用 script 和 link 标签直接引入文件，并使用全局变量 srm
我们在 npm 发布包内的 srm-ui/dist 目录下提供了 srm.js
